# 网盘文件批量重命名插件

一个功能强大的 Tampermonkey 脚本，支持百度网盘、阿里云盘、夸克网盘和 PikPak 云盘的文件批量重命名操作。

## ✨ 功能特性

- 🎯 **多网盘支持**：同时支持百度网盘、阿里云盘、夸克网盘和 PikPak 云盘
- 🔍 **正则表达式**：支持使用正则表达式进行高级文件重命名
- 🔢 **自动编号**：可选择为文件添加序号，支持自定义分隔符和位数
- ⏱️ **延迟控制**：可配置请求延迟，避免触发网盘API限制
- 📊 **批量操作**：支持对当前目录下所有文件进行批量重命名
- 🎨 **美观界面**：现代化的 UI 设计，支持拖拽移动
- 🔄 **自动更新**：内置版本检查和更新提示

## 🚀 安装方法

1. 首先安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [安装脚本](https://update.greasyfork.org/scripts/488421/%E7%BD%91%E7%9B%98%E6%96%87%E4%BB%B6%E6%89%B9%E9%87%8F%E9%87%8D%E5%91%BD%E5%90%8D.user.js) 按钮
3. 确认安装并启用脚本
4. 打开支持的网盘页面，脚本会自动加载

## 📖 使用说明

1. **打开网盘**：访问百度网盘、阿里云盘、夸克网盘或 PikPak 云盘
2. **进入目录**：导航到需要重命名文件的目录
3. **使用面板**：
   - **查找（正则）**：输入要匹配的正则表达式
   - **替换为**：输入替换后的内容
   - **自动编号**：勾选后会为文件添加序号
   - **保留后缀**：勾选后会保留原始文件后缀
   - **延迟(ms)**：设置请求间隔，避免触发API限制
4. **执行操作**：
   - 点击「读取目录」获取当前目录文件
   - 点击「开始改名」执行批量重命名
   - 点击「中止」停止正在执行的操作

## 🎯 正则表达式示例

- **删除方括号内容**：`\[.*?\]`
- **删除多余空格**：`\s+`（替换为单个空格）
- **删除第一个下划线前的内容**：`^[^_]+_`
- **提取数字**：`\D+`（替换为空）

## 🔧 技术特性

- **基于 Vue 3**：使用 Vue 3 构建现代化 UI
- **模块化设计**：清晰的代码结构，易于维护和扩展
- **API 限流处理**：内置请求延迟和错误重试机制
- **安全可靠**：只修改文件名称，不影响文件内容
- **响应式设计**：适配不同屏幕尺寸
