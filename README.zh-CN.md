# PikPak Rename Pro

> 一款可靠的 PikPak 批量重命名工具，无需任何手动验证。

通过浏览器脚本批量重命名 PikPak 文件。

------------------------------------------------------------------------

## 功能

- 批量重命名

- 正则表达式替换

- 自动认证（无需手动输入令牌）

- 可选索引后缀

- 可调节延迟（防速率限制）

------------------------------------------------------------------------

## 安装

1. 安装 Tampermonkey\

2. 添加此脚本\

3. 打开 https://mypikpak.com/

------------------------------------------------------------------------

## 使用方法

1. 打开目标文件夹\

2. 点击页面 → 等待状态变为 `Ready`\

3. 点击 `Scan`\

4. 配置：

- `Search`：正则表达式

- `Replace`：替换（留空 = 删除）

5. 点击 `Rename`

------------------------------------------------------------------------

## 示例

搜索：S01E(\d+)

替换：Episode_$1

------------------------------------------------------------------------

## 注意事项

- 使用前必须达到 `Ready`\

- 建议延迟：1000–2000 毫秒

- 无效的正则表达式将抛出错误

- 大批量数据按顺序执行（设计如此）

------------------------------------------------------------------------

## 开发提示

钩子使用 `fetch` / `XMLHttpRequest` 提取身份验证标头，并使用

`GM_xmlhttpRequest` 进行 API 调用。
